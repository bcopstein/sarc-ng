## Quick Start

### Prerequisites

- Go 1.24+
- Make
- Docker & Docker Compose

### Development Setup

1. **Clone and setup**:

   ```bash
   git clone <repository-url>
   cd sarc-ng
   make setup
   ```

2. **Start development**:

   ```bash
   # Native Go (fastest)
   make run

   # With hot reloading
   make debug

   # Full Docker environment (with database)
   cd docker && docker compose up -d
   ```

## Features

- **Building Management**: Organize physical locations and structures
- **Class Management**: Define and manage classroom spaces
- **Resource Booking**: Reserve and schedule resources
- **Lesson Scheduling**: Plan and organize educational sessions
- **API-First Design**: Comprehensive RESTful API with Swagger documentation

## Core Components

- **REST API**: Access all functionality via HTTP endpoints
- **Database Integration**: MySQL storage with GORM
- **CLI Tools**: Command-line interface for administrative operations
- **Serverless Ready**: AWS Lambda support for cloud deployments
- **Docker Support**: Containerized development and deployment
