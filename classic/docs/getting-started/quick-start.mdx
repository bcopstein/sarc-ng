# Quick Start

Get up and running with SARC-NG in minutes.

## Five-Step Quick Start

### 1. Clone the Repository

```bash
git clone https://github.com/your-organization/sarc-ng.git
cd sarc-ng
```

### 2. Setup Development Environment

```bash
make setup
```

### 3. Start with Docker

The easiest way to get started is with Docker, which sets up the complete environment including database:

```bash
cd docker
docker compose up -d
```

### 4. Verify It's Running

Check that the API is running properly:

```bash
curl http://localhost:8080/health
```

You should see a response like:

```json
{
  "status": "healthy",
  "service": "sarc-ng",
  "timestamp": "2023-06-01T12:34:56Z"
}
```

### 5. Explore the API

Visit the Swagger UI to explore and test the API:

- [http://localhost:8080/swagger/index.html](http://localhost:8080/swagger/index.html)

## Interactive API Example

Try out this simple example of using the API:

```jsx live
function ApiExample() {
  const [data, setData] = useState(null);
  const [loading, setLoading] = useState(false);

  const mockFetchBuildings = () => {
    setLoading(true);
    // Simulate API call
    setTimeout(() => {
      setData([
        {
          id: 1,
          name: "Main Campus Building",
          code: "MCB-A",
        },
        {
          id: 2,
          name: "Science Building",
          code: "SCI-B",
        },
      ]);
      setLoading(false);
    }, 1000);
  };

  return (
    <div
      style={{ padding: "10px", border: "1px solid #ddd", borderRadius: "4px" }}
    >
      <button
        onClick={mockFetchBuildings}
        disabled={loading}
        style={{
          padding: "8px 16px",
          backgroundColor: "#25c2a0",
          color: "white",
          border: "none",
          borderRadius: "4px",
          cursor: "pointer",
        }}
      >
        {loading ? "Loading..." : "Fetch Buildings"}
      </button>

      {data && (
        <div style={{ marginTop: "10px" }}>
          <h4>Buildings:</h4>
          <pre
            style={{
              background: "#f6f8fa",
              padding: "12px",
              borderRadius: "4px",
              overflow: "auto",
            }}
          >
            {JSON.stringify(data, null, 2)}
          </pre>
        </div>
      )}
    </div>
  );
}
```

## Development Workflows

### Native Go Development

For faster development cycles, run directly with Go:

```bash
# Run the application
make run

# With hot reloading
make debug
```

### Build From Source

Build the application:

```bash
make build
```

The binary will be available at `build/bin/app`.
