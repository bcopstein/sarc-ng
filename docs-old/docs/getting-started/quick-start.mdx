# Quick Start Guide

Get up and running with SARC-NG in minutes using this quick start guide.

import ApiEndpointDemo from "@site/src/components/ApiEndpointDemo";

## Prerequisites

Ensure you have:

- [Docker](https://www.docker.com/) and Docker Compose installed
- [Git](https://git-scm.com/) for version control
- [Go](https://golang.org/) (version 1.24 or later) for local development

## 5-Minute Setup

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/sarc-ng.git
cd sarc-ng
```

### 2. Start Docker Environment

```bash
cd docker
docker compose up -d
```

### 3. Access the API

The API is now running at: [http://localhost:8080/api/v1](http://localhost:8080/api/v1)

## Creating Your First Building

### Using cURL

```bash
curl -X POST http://localhost:8080/api/v1/buildings \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Engineering Building",
    "code": "ENG-A"
  }'
```

### Using JavaScript

```javascript
fetch("http://localhost:8080/api/v1/buildings", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    name: "Engineering Building",
    code: "ENG-A",
  }),
})
  .then((response) => response.json())
  .then((data) => console.log("Success:", data))
  .catch((error) => console.error("Error:", error));
```
